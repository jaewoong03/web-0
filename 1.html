HTML

20/07/20 오늘부터 HTML 문법을 정리해 보도록 하겠다

* A-코드는 Attribute 코드로 대상의 속성을 지정하는 코드이다 *

1일차 7/20

  • br 코드 : 이 코드는 단순히 줄 바꿈을 할때 사용하는 코드이다. 단순히 줄바꿈을 하는것 뿐이기 때문에 다른 tag들과는 다르게 마지막에 /br이 붙지 않는다.
  • p 코드 : 이 코드는 '문단'을 바꿀때 사용하는 코드이다 br 코드와는 다르게 문단을 변경하는 것 이기 때문에 사용해본 결과 두줄을 띄게 된다. 웹코딩을 하는 사람들 사이에서는 br 코드보다는 p코드를 사용하는게 더 편리하고 유익하다고 한다.
  • h1, h2, h6 코드 : 이 코드는 제목을 지을때 사용한다. h 다음에 붙은 숫자가 작을수록 큰 제목이고 커질수록 부제목이라고 생각하면 된다.
  • img 코드 : 말그대로 이미지를 삽입할때 사용하는 코드이다.
  • A-src 코드 : 외부 리소스의 '위치'를 지정할때 사용하는 코드이다.
  • A-width 코드 : 소스의 넓이를 픽셀단위로 지정하는 코드이다.

2일차 7/21

  • li 코드 : 리스트, 즉 목록을 만들때 사용하는 태그이며 반드시 단독으로 쓰이지 않으면 다른 태그 내부에 들어가 사용된다
  • ul 코드 : 목록을 정의하는 태그로 반드시 내부에 다른 태그가 들어있어 함께 사용되어야 한다
  • ol 코드 : 정렬된 목록을 정의하는 태그로 목록을 숫자 또는 알파벳순으로 정렬한다.

3일차 8/05

  • head 태그 : 말머리들을 묶는 태그
  • body 태그 : 본문들을 묶는 태그
  • !DOCTYPE html : 아래 문서가 html 임을 나타내는 태그
  • html 태그 : 모든 태그들을 묶는 최고위 태그
